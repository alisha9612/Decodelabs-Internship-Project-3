## Project 3

## Project Overview

The objective of this project is to transform low-fidelity wireframes into a high-fidelity user interface for a premium organic food delivery application. The design focuses on creating a visual identity that communicates trust, health, freshness, and convenience through thoughtful use of colors, typography, imagery, and layout.

## Screens Designed

### 1. Splash Screen
- Brand introduction
- Welcoming experience
- Clear call-to-action

### 2. Product List Screen
Features:
- Personalized greeting
- Search bar
- Promotional banner
- Product categories
- Fresh Picks section
- Bottom navigation

### 3. Product Details Screen
Features:
- Product hero image
- Product information
- Price and rating
- Quantity selector
- Product description
- Similar product recommendations
- Add to Cart and Buy Now actions

## Visual Identity

| Secondary Text | `#757575` |

### Typography

- **Headings:** Poppins
- **Body Text:** Poppins

The typography was chosen to create a clean, modern, and user-friendly experience.

---

## ✨ Design Decisions

- Used green tones to represent freshness, health, and trust.
- Applied rounded cards and soft shadows to create a friendly and approachable interface.
- Incorporated high-quality product imagery to enhance visual appeal.
- Maintained consistent spacing and alignment across all screens.
- Added recommendation sections and prominent call-to-action buttons to improve usability and engagement.
